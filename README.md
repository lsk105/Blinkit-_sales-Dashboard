# Blinkit Sales Dashboard | Power BI
# Project Overview

This project presents an interactive Power BI dashboard developed using Blinkit grocery sales data. 
The objective of the dashboard is to analyze sales performance, product trends, outlet characteristics, and customer ratings to generate actionable business insights and support data-driven decision-making.

# Business Problem

Retail businesses generate large volumes of transactional data that can be difficult to interpret without effective reporting. 
This dashboard transforms raw sales data into meaningful visualizations, enabling stakeholders to monitor performance, identify trends, and make informed business decisions.

# Dashboard Features
# 1. Sales Performance Overview

Provides a high-level summary of overall sales performance using KPI cards.
2. Item Fat Content Analysis
Analyzes sales distribution between Low Fat and Regular products to understand customer purchasing preferences.
3. Item Type Analysis
Evaluates sales performance across multiple product categories such as:
Fruits & Vegetables
Snacks
Household Items
Frozen Foods
Dairy Products
Canned Goods
Beverages
4. Outlet Establishment Analysis
Tracks sales trends based on outlet establishment years to identify business growth patterns over time.
5. Outlet Size Analysis
Compares sales performance across Small, Medium, and High-sized outlets.
6. Outlet Location Analysis
Analyzes sales contribution across Tier 1, Tier 2, and Tier 3 locations.
7. Outlet Type Analysis
# Compares different outlet types based on:
Total Sales
Average Sales
Average Rating
Number of Items
Item Visibility
8. Interactive Filters

Users can dynamically filter the dashboard using:

Outlet Location Type
Outlet Size
Item Type
Key Insights
